My Proxmox Homelab: A Professional-Grade IT Environment

As a System Engineer and IT Consultant, I make it a priority to work hands-on with the technologies I recommend. This homelab is my way of keeping skills sharp, testing new ideas, and ensuring I can deliver reliable solutions in real-world situations.

It is built to reflect professional infrastructure standards, with a focus on scalability, uptime, and security. Every component is there for a reason and is managed as if it were running in a live production environment.

> Core Infrastructure: Proxmox VE Cluster

  • High Availability (HA) to keep services online
  • Live migration between nodes for maintenance and upgrades
  • Centralized cluster management
  • ZFS-backed storage for speed and data protection

This cluster setup makes it possible to run complex workloads and test advanced scenarios such as rolling upgrades, service migrations, and load balancing.

> Storage: Dual TrueNAS SCALE NAS Servers

  • Two TrueNAS SCALE systems for redundancy and scalability
  • ZFS-based storage with snapshots and replication
  • Separate storage pools for different workloads
  • Integration with Proxmox VE for shared storage

By using two TrueNAS SCALE servers, I can replicate data between sites and maintain consistent, high-performance storage for all virtual machines and containers.

> Reliable Backup and Disaster Recovery

My backup strategy uses both on-site and remote Proxmox Backup Servers for secure, deduplicated, and encrypted backups.

  • Disaster recovery across different locations
  • Fast incremental restores
  • Snapshot-based backups for virtual machines and containers

> Networking and Zero Trust Security

  • Zero Trust Network Access powered by NetBird
  • MikroTik routers and switches for reliable, flexible networking
  • Reverse proxy with HTTPS
  • Identity-based authentication via a self-hosted IdP
  • Network segmentation with VLANs and firewalls
  • Site-to-site VPN for secure remote access
  • Continuous monitoring and alerting

Security is part of the design from the ground up, ensuring every service and device is protected.

These services are fully integrated, monitored, and secured, following the same standards I apply in client environments.

  • Nextcloud – File sync, calendar, and contact management

  • Pocket ID – Identity provider for single sign-on

  • Servarr Stack – Automated media library management
