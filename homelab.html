Homelab

As a System Engineer and IT Consultant, I keep skills sharp by working hands-on with the technologies I recommend. This homelab is built to reflect professional infrastructure standards with a focus on scalability, uptime, and security.

Core Infrastructure

Proxmox VE cluster providing:

  • High Availability to keep services online
  • Live migration between nodes for maintenance
  • Centralized cluster management
  • ZFS-backed storage for speed and data protection

Storage

Dual TrueNAS SCALE NAS servers:

  • ZFS-based storage with snapshots and replication
  • Separate storage pools for different workloads
  • Integration with Proxmox VE for shared storage

Backup & Recovery

Using on-site and remote Proxmox Backup Servers for secure, deduplicated, encrypted backups:

  • Disaster recovery across locations
  • Fast incremental restores
  • Snapshot-based VM and container backups

Networking & Security

  • Zero Trust Network Access via NetBird
  • MikroTik routers and switches
  • Reverse proxy with HTTPS
  • Self-hosted IdP for identity-based auth
  • Network segmentation with VLANs and firewalls
  • Site-to-site VPN for secure remote access
  • Continuous monitoring and alerting

Self-Hosted Services

Fully integrated, monitored, and secured:

  • Nextcloud - File sync, calendar, contacts
  • Jellyfin - Private media streaming
  • Vaultwarden - Password management
  • Pocket ID - Single sign-on
  • Home Assistant - Smart home automation
  • Immich - AI photo backup
  • Servarr Stack - Media library management

  • Compare self-hosted vs cloud options
