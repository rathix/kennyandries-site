> About me

I am a system engineer from Belgium with a focus on secure, automated, and reliable IT infrastructure. At NMBS/SNCB I manage large scale mobile device deployments with Microsoft Intune, Samsung Knox, and SureMDM, and I run core cloud services such as Microsoft Exchange and Entra ID.

My path moved from hands on support to system administration to senior engineering. I know the Microsoft enterprise stack well and I put a lot of effort into automation with PowerShell, Ansible, and Terraform. I work with Docker and Podman for container deployments and I enjoy Linux, immutable infrastructure, and server hardening. NixOS and Fedora Atomic are favorites.

Outside of work I run a personal homelab where I test new ideas, refine automation workflows, and look for ways to improve performance and security. I also offer consulting services for teams that want to improve MDM, Microsoft 365, and DevOps practices. If you want to talk about a project, get in touch.
